# ❌⭕ Tic Tac Toe

A simple 2-player **Tic Tac Toe game** made with Python.

## 🎮 Features
- 3x3 grid board  
- Player X and Player O turns  
- Win/draw detection  
- Console-based version (expandable to GUI later)  

## ▶️ How to Run
```bash
python tictactoe.py
